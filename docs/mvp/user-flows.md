# Primary user flow

User signs in → lands on dashboard → creates workspace → adds members → views activity / usage → updates settings.

---

# Admin flow

Admin signs in → opens tenant settings → configures RBAC → manages integrations → reviews platform activity.

---

# Operational flow

Engineer/admin opens observability page → checks latency/errors/background jobs → drills into logs/events.

---

# Out of scope for MVP

- Advanced billing  
- Multi-region DR automation  
- Full audit export  
- Complex analytics  
- AI recommendations  
- Mobile app  

That out-of-scope list matters because MVP references consistently say the exclusions are what protect the scope from bloating.
